Privacy Policy for PDFora

Last Updated: 11 March 2026

This Privacy Policy describes how PDFora handles user information. PDFora is a web-based PDF reader and editor developed by CROOKS. The application allows users to open, view, read, and edit PDF documents stored on their device. We respect user privacy and are committed to protecting it.

1. Information Collection

PDFora does not collect, store, transmit, or process any personal information from users. The application does not require account registration or personal details such as name, email address, phone number, or location information.

2. Device Storage Access

PDFora may request permission to access files stored on the user's device. This permission allows users to open, read, edit, and save PDF files using the device file manager. Access occurs only when the user selects a file or saves an edited document.

3. Local File Processing

All PDF viewing and editing operations are performed locally on the user's device. Files opened or edited in the application remain on the device unless the user chooses to export or share them using device features.

4. Data Usage

Since PDFora does not collect personal information, the application does not use, store, analyze, or process any personal or sensitive user data.

5. Data Sharing

PDFora does not share user data with third parties because the application does not collect any personal information.

6. Third-Party Services

PDFora does not use third-party SDKs, advertising services, analytics platforms, or tracking technologies.

7. Security

All document processing occurs locally on the user's device. No files or personal information are transmitted to external servers while using the application.

8. Children's Privacy

PDFora does not knowingly collect personal information from children under the age of 13. Since the application does not collect data, it is suitable for general audiences.

9. Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. Any updates will be posted on this page with a revised update date.
